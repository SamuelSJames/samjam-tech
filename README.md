## GitHub Pages
This repo is published with GitHub Pages from the `main` branch root.

If needed, in GitHub:
1. Open **Settings**
2. Open **Pages**
3. Set **Source** to **Deploy from a branch**
4. Select **main** and **/(root)**
5. Save
