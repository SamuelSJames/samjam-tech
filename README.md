# samjam-tech

Simple GitHub Pages resume site for Samuel James.

## Files
- `index.html` - current resume page
- `style.css` - styling
- `.nojekyll` - disables Jekyll processing for static hosting

## Publish on GitHub Pages
In GitHub:
1. Open **Settings**
2. Go to **Pages**
3. Set source to **Deploy from a branch**
4. Select **main** and **/(root)**
5. Save

Your site should publish at the repo Pages URL after the first push.
